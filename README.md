# Manuscript Storage Warehouse（手稿存储仓库）

这是一个个人手稿与随笔的集合，主要围绕 Linux、操作系统设计与使用心得的中文文章（以及其中一篇的英文译文）。适合对操作系统思想、Linux 入门与进阶话题感兴趣的读者阅读与讨论。

## 仓库内容（顶层文件）
[] Linux杂谈之我和Linux.md — 作者与 Linux 的故事与感想。
[] linux-is-not-god.md — 关于对 Linux 的反思（中文）。
[] linux-is-not-god_eng.md — 上文的英文译文。
[] 什么是“原子系统”？.md — 对“原子系统”概念的阐述与讨论。
[x] 如何向他人推荐一个“不完美”的系统.md — 交流和推荐复杂系统时的策略与建议。
[] 我理想中的操作系统，是什么样的？.md — 对理想操作系统特性的设想与思考。
[] 正确入门linux指南.md — 面向初学者的 Linux 入门建议与路径。

> 注：文件名即为仓库中文稿件的文件名，仓库以 Markdown 文件为主，直接在 GitHub 上打开即可阅读。

## 如何阅读与使用
1. 克隆或直接在 GitHub 页面浏览。文件均为 Markdown 格式，适合直接在线阅读或下载离线查看。 

   git clone https://github.com/fanh211/Manuscript_Storage_Warehouse.git

2. 若想贡献修订或翻译：
   - Fork 本仓库并在新分支上修改对应的 Markdown 文件。
   - 提交 Pull Request，说明修改或翻译的目的与范围。

3. 如果想把某篇文章导出为 PDF，可使用支持 Markdown 转 PDF 的工具（如 pandoc、Typora 等）。
